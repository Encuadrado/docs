Encuadrado Web is a full-stack web application that powers a healthcare scheduling and practice management platform. It enables healthcare professionals to manage bookings, patients, payments, prescriptions, and clinical records.

## Technology stack

| Category | Technologies |
|----------|-------------|
| **Backend** | Python 3.12, Flask 3.x, SQLAlchemy 2.x ORM |
| **Database** | PostgreSQL |
| **Frontend** | React 18, Webpack, Jinja2 templates, vanilla JS |
| **Background jobs** | Celery with Redis broker |
| **Containerization** | Docker with docker-compose |
| **Infrastructure** | AWS Elastic Beanstalk, RDS, S3, CloudFront |
