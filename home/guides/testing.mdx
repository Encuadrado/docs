## Setup

The project uses Vitest with Testing Library and MSW (Mock Service Worker) for API mocking.

**Configuration**: `vitest.config.ts`
- Environment: jsdom
- Setup file: `vitest.setup.ts` (mocks for lottie-web and react-i18next)

## Running tests

```bash
pnpm test          # Watch mode
pnpm test:run      # Single run
pnpm test:ui       # Vitest UI
```
