Encuadrado Home is the professional-facing dashboard where healthcare practitioners manage their day-to-day operations: scheduling, patients, billing, prescriptions, team management, and more. It's a Next.js application mounted at `/cuenta` that communicates with the Flask backend (Encuadrado Web).

## Technology stack

| Category | Technologies |
|----------|-------------|
| **Framework** | Next.js 15, React 19, TypeScript |
| **Styling** | Tailwind CSS 4, Radix UI components |
| **State** | TanStack Query v5, React Context API |
| **Forms** | React Hook Form + Zod validation |
| **Testing** | Vitest, Testing Library, MSW |
| **Component docs** | [storybook.encuadrado.com](https://storybook.encuadrado.com/) (centralized Storybook) |
| **Code quality** | Biome (linter + formatter) |
| **Error tracking** | Sentry |
| **Package manager** | pnpm |
| **Runtime** | Node 22.x |
